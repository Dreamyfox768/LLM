# LLM
Learn how to download ollama
1- go to ollama website 
2- hit download 
3- download according to os 
4- go to shell/terminal
5- write ollama to make sure it is dowloaded
6- to download specific version, check ollama github to make sure you have enough RAM 
7- write ollama run __version__
8- to exit out write /bye in terminal
9- to see what model of ollama you have type ollama list
