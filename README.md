# Ollama LLM Download 

Learn how to download Ollama

****

## Steps to Download and Run Ollama

- Go to Ollama website  

- Hit download  

- Download according to OS  'window, mac, or linux'

- Go to cmd/shell/terminal  

- Write `ollama` to make sure it is downloaded  

- To download specific version, check Ollama GitHub to make sure you have enough RAM  

- Write `ollama run <version>`  

- To exit out write `/bye` in terminal  

- To see what model of Ollama you have, type `ollama list`
